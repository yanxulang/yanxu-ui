# 变更日志

## [0.9.0] - 2026-07-22

### 1.0 候选冻结

- 冻结包含 30 个声明、23 个类、7 个包级函数、125 个域和 375 个方法的公开 API；当前
  快照与 1.0 候选冻结保持逐字节一致。
- 增加 0.1.0–0.8.0 十个已发布 API 基线的机器兼容门禁，并正确识别由子类上移到父类的
  继承成员。
- 提供逐版本迁移证明，明确言台 1.0 锁、权限、托管资源所有权、关闭报告和集合无障碍语义
  的升级处理方式。
- 冻结言台、言据、最低工具链锁、许可证、工作流动作、源码包导入和自有原生库的供应链证据。

### 发布候选

- 在六目标汇总后重新下载四个发布附件，验证外部摘要、清单、API 冻结、归档安全路径以及
  归档内兼容和供应链证据。
- 逐目标核对根锁、示例锁、言序 1.1.20 运行时元数据、自动关闭 YXB 内部摘要与字节数，并
  上传可保留的候选演练报告。
- 新增 0.x–1.0 兼容证明、迁移指南、0.9 发布说明和可本地复现候选演练的生产验收文档。

## [0.8.0] - 2026-07-22

### 高级控件与无障碍

- 滚动容器增加内容范围夹取和合法`滚动`动作；普通／虚拟列表生成最多 256 个当前可见
  `列表项`节点，并把选择和滚动到动作归属到项目节点。
- 标签页生成稳定`标签`节点并增加程序选择 API；分割面板统一指针、键盘、程序与辅助技术
  调整；弹出层支持辅助展开和折叠。
- 所有控件语义值规范为言台 1.0 支持的标量，集合诊断改由公开查询 API 提供；内部编号和
  动作映射不进入公开签名。

### 恢复与生产验收

- 识别原生 ABI 错误格式，并在言台已经回收资源时把后续幂等关闭视为成功收敛。
- 修复虚拟列表重复应用滚轮增量，并以精确滚动值和数据源行选择回归固定事件路由。
- 将滚动、列表语义、标签页和分割面板算法移入独立内部模块，保持包入口低于 32 MiB 上限。
- 新增六目标生产后端、全控件无障碍、4096 轮交互和 1024 轮资源归零验收；发布候选覆盖
  14 项单元测试、11 条无窗口集成、5 条真实窗口路径和 16 个 Release YXB。
- 新增生产验收指南，冻结 0.8 API JSON 与完整参考文档。

## [0.7.0] - 2026-07-22

### 生命周期与资源

- 在创建首个子资源前验证言台 1.0 配额、拒绝统计、退出幂等、生命周期统计和资源归零能力，
  并公开硬上限复制值、当前配额与冻结前下调入口。
- 将计时器升级为托管任务：一次任务自动完成，回调错误隔离，取消／关闭幂等，结束后解除
  应用登记并忽略迟到事件。
- 将图片升级为托管资源：校验尺寸和持有字节，失败创建自动回滚，关闭后归还配额并置空绘制
  句柄；应用关闭统一清理仍活动的图片。

### 关闭、诊断与验证

- 窗口和应用使用可用、关闭中、已关闭的单向状态，按步骤最佳努力清理并返回可 JSON 序列化
  的成功数和结构化错误；重复关闭不会再次执行。
- 诊断快照升级到版本 2，增加生命周期报告和活动图片资源数；应用关闭后不再查询已释放句柄。
- 新增 128 轮窗口／计时器／图片资源归零压力测试、生命周期公开示例和完整恢复指南，并在
  最低与当前工具链以及六目标矩阵执行。
- 生命周期编排和资源登记移入独立模块，包入口与 15 个公开示例共 16 个 Release YXB 继续
  满足单模块 32 MiB 安全上限。

## [0.6.0] - 2026-07-22

### 动画与渲染

- 增加有界动画任务、四种缓动、完成回调、取消和生命周期统计；每窗最多 1024 个活动任务。
- 使用言台帧提交回执和`帧呈现`事件推进动画，以单槽待呈现身份处理背压、替换和迟到反馈。
- 更新或完成回调失败只终止当前任务；回调内取消任务、取消队列或关闭窗口不会破坏遍历状态。
- 动画到达终值后提交静态终帧，窗口关闭时取消活动任务并停止访问已释放平台资源。

### 诊断与性能

- 增加版本化应用与窗口诊断快照，汇总布局、交互、无障碍、帧、动画和平台运行状态，且不
  复制窗口标题、显示器名称、控件内容、回调或原生句柄。
- 增加 Linux x86-64 Release 性能预算，覆盖动画生命周期、帧反馈、诊断快照和布局／帧编码。
- 新增动画与诊断公开示例、指南、API 快照及最低／当前工具链回归门禁。

## [0.5.0] - 2026-07-22

### 文本与数据视图

- 增加 Unicode 字节边界、行范围和按行选区 API；多行输入框按可见行绘制精确选区。
- 增加有界增量数据源、缓存淘汰、加载失败诊断、版本事件和虚拟列表控件。
- 增加数据视图集成回归、公开示例、指南和 API 导出；不改变言台 1.0 依赖或 ABI。
- 列表与虚拟数据视图实现拆到独立模块，最低工具链构建继续满足单模块 32 MiB 安全上限。

## [0.4.0] - 2026-07-22

### 叠层与菜单

- 新增有界弹层放置、四方向自动翻转、视口夹取和窗口缩放后的重新布局。
- 新增窗口级叠层栈、逆序命中、外部点击关闭、Escape 关闭和嵌套模态焦点范围。
- 下拉选择接入统一叠层；托管叠层在根布局中保留锚框，避免一次重绘被归零。
- 菜单支持 1024 项单层、4096 项总量、16 层深度和标题／动作输入校验；支持子菜单、禁用
  项、分隔项、指针命中、完整方向键导航和稳定无障碍动作。

### 文档与打包

- 新增叠层与菜单指南、子菜单示例、真实窗口回归和完整 API／架构说明。
- 将输入框键盘编辑处理移入表单实现模块，使作为依赖包归档时入口模块保持在 32 MiB
  生产上限内；不放宽 YXB 安全门禁。
- API 快照、参考文档、六目标候选和 12 个公开示例继续由言序 1.1.9 生成并由 1.1.20
  复核。

## [0.3.0] - 2026-07-22

### 表单

- 新增复选框、单选框、切换、下拉选择、横纵滑块以及确定/不定进度条。
- 新增只读、必填、无效通用状态；只读输入保留焦点、选择和复制，同时拒绝交互改值。
- 下拉选择限制为 4096 项和单项 65536 UTF-8 字节；滑块与进度条拒绝非有限范围和值。

### 无障碍

- 接通可检查控件、组合框和滑块的原生动作回传，并在真实窗口中验证树修订和值同步。
- 进度条以无动作输出角色公开确定值，或以空值和忙碌状态表达不定进度。

### 文档与持续集成

- 新增完整表单指南与可构建示例；六目标候选现在验证 12 个示例和包入口共 13 个 YXB。
- API 快照与参考文档由最低支持的言序 1.1.9 重新生成，1.1.20 继续验证兼容与正式构建。

## [0.2.0] - 2026-07-21

### 主题

- 新增原子运行时主题更新、样式缓存失效、修订号和全窗口重新布局。

### 无障碍

- 语义快照使用控件稳定编号并输出明确的可见、启用、可聚焦、焦点和操作状态。
- 窗口把脏语义树提交到言台 1.0 原生桥，并在关闭前清理；查询可返回规范化树和修订摘要。
- 焦点与动作请求重验树修订和当前控件身份；按钮点击及输入框设置值、UTF-8 选区、复制、
  粘贴已接通。

### 修复

- 图层命令改用言台绘制协议规定的`透明度`字段，禁用控件可在真实窗口中正常编码和绘制。

### 发布

- 六目标自动关闭应用改由言序 1.1.20 在隔离源码副本中构建并立即回放，发布候选不再只做
  编译存在性检查；公开的 1.1.9 目标锁和源码兼容门禁保持不变。
