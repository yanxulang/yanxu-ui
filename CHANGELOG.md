# 变更日志

## [0.3.0] - 2026-07-22

### 表单

- 新增复选框、单选框、切换、下拉选择、横纵滑块以及确定/不定进度条。
- 新增只读、必填、无效通用状态；只读输入保留焦点、选择和复制，同时拒绝交互改值。
- 下拉选择限制为 4096 项和单项 65536 UTF-8 字节；滑块与进度条拒绝非有限范围和值。

### 无障碍

- 接通可检查控件、组合框和滑块的原生动作回传，并在真实窗口中验证树修订和值同步。
- 进度条以无动作输出角色公开确定值，或以空值和忙碌状态表达不定进度。

### 文档与持续集成

- 新增完整表单指南与可构建示例；六目标候选现在验证 12 个示例和包入口共 13 个 YXB。
- API 快照与参考文档由最低支持的言序 1.1.9 重新生成，1.1.20 继续验证兼容与正式构建。

## [0.2.0] - 2026-07-21

### 主题

- 新增原子运行时主题更新、样式缓存失效、修订号和全窗口重新布局。

### 无障碍

- 语义快照使用控件稳定编号并输出明确的可见、启用、可聚焦、焦点和操作状态。
- 窗口把脏语义树提交到言台 1.0 原生桥，并在关闭前清理；查询可返回规范化树和修订摘要。
- 焦点与动作请求重验树修订和当前控件身份；按钮点击及输入框设置值、UTF-8 选区、复制、
  粘贴已接通。

### 修复

- 图层命令改用言台绘制协议规定的`透明度`字段，禁用控件可在真实窗口中正常编码和绘制。

### 发布

- 六目标自动关闭应用改由言序 1.1.20 在隔离源码副本中构建并立即回放，发布候选不再只做
  编译存在性检查；公开的 1.1.9 目标锁和源码兼容门禁保持不变。

## [0.1.2] - 2026-07-21

### 修复

- 重构指针和键盘事件的目标选择，兼容言序 1.1.15–1.1.20 对初始空值的严格类型推断。

### 依赖

- 将言台从 0.1.0 升级到稳定版 1.0.0，继续使用 ABI v2 和既有协议组合，并改用其六目标
  正式制品与 1.x 支持政策。

### 持续集成

- 在六目标最低言序 1.1.9 验证之外，增加当前稳定言序 1.1.20 的源码、测试、集成和正式
  构建门禁。
- 为言台 1.0 冷加载设置每卷 30 秒硬上限，避免默认 10 秒误报，同时继续拒绝无界挂起。

## [0.1.1] - 2026-07-16

### 优化

- 主题对象按样式名称与悬停、按下、焦点、禁用状态位缓存解析结果。
- 新增`控件样式`合并缓存，避免每个控件在每帧重复深合并基础样式与种类样式。

### 运行时

- 最低言序版本推进到 1.1.9，使 GUI、计时器和原生扩展宿主回调使用彼此隔离的执行步数预算。

## [0.1.0] - 2026-07-15

### 新增

- 建立言序 1.1.8 保留模式 GUI 包骨架，并固定 Windows 图形回调栈修复。
- 增加几何约束、脏区、主题配置和可撤销文本模型。
- 增加行、列、堆叠、网格、滚动和分割布局。
- 增加事件捕获、目标、冒泡、焦点、快捷键和指针捕获。
- 增加保留渲染树、二进制整帧提交、状态绑定、动画和语义树。
- 增加文字、按钮、单／多行输入、列表、标签页、菜单、弹出层、画布和图片控件。
- 增加言据优先与 JSON 兼容的主题配置。
- 增加十一个功能示例、真实窗口冒烟和跨平台发布门禁。
