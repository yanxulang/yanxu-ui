# 示例索引

| 文件 | 验证内容 |
| --- | --- |
| `最小窗口.yx` | 应用、窗口、文字和关闭 |
| `按钮和布局.yx` | 行列布局、按钮状态和点击 |
| `文本输入.yx` | 单行、多行、变化事件 |
| `中文输入法.yx` | 中文 IME、Emoji、扩展 Unicode |
| `滚动列表.yx` | 滚动容器和列表选择 |
| `多窗口.yx` | 单事件循环多原生窗口 |
| `绘制展示.yx` | 画布和整帧绘制命令 |
| `主题切换.yx` | 运行时浅／深主题 |
| `言据配置.yx` | 言据主题加载（同 API 支持 JSON） |
| `综合控件展示.yx` | 叠层菜单、子菜单、分割、列表、标签页、输入和画布 |
| `完整表单.yx` | 表单状态、可检查控件、选择、滑块、进度和提交校验 |
| `自动关闭冒烟.yx` | CI 中真实窗口自动退出 |

从工作区根目录执行类型检查：

```sh
for file in yanxu-ui/examples/*.yx; do
  yanxu-language-new/target/debug/yanxu 查 "$file"
done
```

`examples/言序.toml`声明图形、原生扩展、剪贴板和文件对话框权限；其入口是可自动退出的
冒烟程序。修改包内容后应重新生成该目录的目标平台`言序.lock`。
