# 叠层与菜单

言界的菜单、下拉选择和弹出层共享窗口级叠层栈。叠层只在窗口绘制和事件循环内存在，
不创建额外原生窗口；窗口关闭时会按逆序清理所有已登记叠层。

## 创建菜单

容器使用`菜单（项目列）`创建菜单按钮。项目可以是文字，也可以是典：

```yanxu
定 文件菜单 为 工具栏.菜单（【
    {「标题」：「文件」，「快捷键」：「⌘N」，「子菜单」：【
        {「标题」：「新建」}，
        {「标题」：「打开」，「快捷键」：「⌘O」}
    】}，
    {「标题」：「退出」，「禁用」：假}
】）；
文件菜单.监听（「选择」，处理选择）；
```

项目典支持以下字段：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `标题` | 文 | 项目显示文字；文字项目会自动转换为此字段 |
| `子菜单` | 列 | 非空时显示子菜单箭头，并可用右方向键打开 |
| `快捷键` | 文 | 仅用于显示，不自动注册全局快捷键 |
| `禁用` | 理 | 禁用项目不可命中、不可键盘选择、不会触发动作 |
| `分隔` | 理 | 绘制分隔线；分隔项目不可选择 |
| `动作` | 法 | 叶项目激活前调用，参数为选择数据典 |
| `保持打开` | 理 | 叶项目激活后保持菜单展开；默认关闭菜单 |

输入会在创建时校验：单层最多 1024 项、总数最多 4096 项、最多 16 层子菜单，单个标题
最多 65536 个字符。无效项目会抛出稳定的`UI_MENU_*`错误，而不会生成部分菜单。

## 打开、关闭与导航

菜单提供`打开（）`、`关闭菜单（）`、`切换（）`、`取展开（）`和`选择（序）`。指针按下
菜单按钮即可打开；点击菜单外部、按`Escape`或激活叶项目会关闭。打开时窗口叠层栈会把
菜单置于普通控件之上，子菜单区域不会被误判为外部点击。

键盘行为如下：

| 按键 | 行为 |
| --- | --- |
| `Up`／`Down` | 在当前层跳到上一个／下一个可选项目，跳过禁用和分隔项目并循环 |
| `Home`／`End` | 跳到当前层首个／末个可选项目 |
| `Right` | 打开当前项目的子菜单 |
| `Left` | 返回父菜单层；根层不关闭菜单 |
| `Enter`／空格 | 激活当前项目，或打开其子菜单 |
| `Escape` | 关闭当前子菜单层；根层关闭整个菜单 |

子菜单锚定在父项目行旁边；空间不足时自动翻转到另一侧，并按视口边距夹取。窗口缩放
或根布局发生时，所有打开层会重新计算位置。叠层托管控件在根布局中保留自己的锚框，避免
菜单或下拉在一次重绘中被布局算法归零。

## 选择事件

叶项目激活时触发`选择`事件，事件数据包含：

```text
{
  「序」：当前层项目序号，
  「层」：当前子菜单层级，
  「项目」：规范化后的项目典，
  「路径」：从根层到当前项目的序号列
}
```

如果项目包含`动作`函数，会先以该数据典调用函数，再发送`选择`事件。动作抛出的错误会
向调用方传播；菜单状态不会留下半个已激活层。`保持打开`只影响叶项目成功后的关闭行为。

## 无障碍

展开菜单的语义根角色为`菜单`，每层项目输出`菜单项`子节点；分隔线输出`分隔符`。节点
包含稳定编号、边界、`选中`、`启用`和`展开`状态。子菜单层作为对应父项目的语义子树输出，
因此屏幕阅读器可以在同一棵修订树中浏览层级。

菜单声明`展开`、`折叠`和`聚焦`动作；可选项目声明`点击`，带子菜单项目还声明相应的
`展开`或`折叠`。平台动作请求必须携带当前树修订，言界会再次确认项目仍可见、启用且未
关闭，拒绝过期编号、禁用项目和非法参数。应用也可以直接调用：

```yanxu
定 快照 为 窗口.无障碍快照（）；
窗口.处理（{「类型」：「无障碍动作请求」，「树修订」：快照【「修订」】，
    「节点」：目标编号，「动作」：「点击」，「参数」：空}）；
```

`叠层快照（）`只提供菜单层级、选择和已夹取边界的诊断副本；它不是原生句柄，也不应跨
窗口保存。

## 错误与边界

常见稳定错误码包括`UI_MENU_ITEMS`、`UI_MENU_ITEM`、`UI_MENU_DEPTH`、`UI_MENU_TITLE`、
`UI_MENU_SUBMENU`、`UI_MENU_DISABLED`、`UI_MENU_SEPARATOR`、`UI_MENU_SHORTCUT`和
`UI_MENU_ACTION`。菜单不负责注册快捷键，也不承诺 GPU、移动端或 Web 后端；快捷键绑定仍
由窗口快捷键表负责，绘制和原生桥能力由已发布的言台 1.0 契约提供。

完整可运行示例见[`examples/综合控件展示.yx`](../examples/综合控件展示.yx)，叠层命中、
翻转、外部关闭、键盘导航和无障碍动作回归见[`integration/菜单窗口.yx`](../integration/菜单窗口.yx)。
