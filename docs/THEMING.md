# 样式、主题与配置格式

言界内置浅色和深色主题。样式先解析继承，再依次叠加`悬停`、`按下`、`焦点`、`禁用`
状态，最后叠加控件实例的`样式`配置。主题初始化后，解析样式和“控件基础样式 + 种类样式”
按种类与四个状态位缓存；控件实例配置仍在最后叠加。循环继承会返回`UI_STYLE_CYCLE`。

## 言据（首选）

```yanju
据【
  「版本」：1，
  「名称」：「海蓝」，
  「基于」：「浅色」，
  「颜色」：据【「主色」：列【20，112，210，255】】，
  「度量」：据【「圆角」：10】，
  「样式」：据【
    「按钮」：据【
      「背景」：列【20，112，210，255】，
      「前景」：列【255，255，255，255】
    】
  】
】
```

```yanxu
引「包:言界/配置」为 配置；
定 数据 为 配置.解析配置（原文，「言据」）；
应用.主题配置（配置.规范主题（数据））；
```

## JSON（兼容）

```json
{"版本":1,"名称":"海蓝","基于":"浅色","颜色":{"主色":[20,112,210,255]},"度量":{},"样式":{}}
```

把格式参数改为`JSON`即可；`自动`先尝试言据，再尝试 JSON。`输出配置`和主题对象的
`言据／JSON`方法可做稳定往返。配置文本只用于启动、交换和调试；事件与每帧绘制不会使用
言据或 JSON。

主版本当前为 1。未知可选字段会保留；主版本不同会返回`UI_CONFIG_MAJOR`。
