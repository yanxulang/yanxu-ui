# 言界 0.9.0

言界 0.9.0 是 1.0 的候选冻结版本。它不新增公开控件 API，而是把 0.1.0–0.8.0 的源码
兼容、当前公开面、直接依赖、许可证和六目标发布候选转为可重复生成、逐字节比较的机器证据。

## 新功能

- 冻结 1.0 候选公开面：30 个声明、23 个类、7 个包级函数、125 个域和 375 个方法；当前
  API 与冻结快照的 SHA-256 均为
  `60596fd0f1c6e4a89cb44acd6109fe40945d386fa617d8f562ee8e8486c1a228`。
- 新增逐标签兼容门禁，从 `v0.1.0` 到 `v0.8.0` 的十个已发布 API 基线验证声明、继承、域、
  参数和返回类型；兼容报告随仓库和候选归档发布。
- 新增供应链冻结，逐项核对清单、最低工具链锁、言台与言据精确提交、许可证摘要、固定提交
  的工作流动作、源码包导入以及自有原生库边界。
- 新增发布候选演练，重新下载四个待发布附件并验证外部摘要、归档路径、六目标锁、运行时
  元数据、内部 YXB 摘要和归档内冻结证据。

## 修复内容

- 兼容检查会沿父类查找旧域和旧方法，避免把公开实现上移到父类误判为删除。
- 候选校验同时支持 Linux `sha256sum`和 macOS `shasum -a 256`，本地复现与 CI 使用同一
  判定逻辑。
- 候选归档中的越界路径、链接、特殊文件、附件替换、目标锁错配和运行时目标错配现在都会
  在进入 Release 前失败。

## 兼容性与平台

- 最低源码工具链仍为言序 1.1.9；正式制品仍由言序 1.1.20 在隔离源码副本中构建并回放。
- 言台固定为 `v1.0.0`，言据固定为提交 `765d9dd623db901a3e71aa4759dbcd77563cb3a9`；
  ABI v2、平台 1.7、事件 1.3、无障碍 1.0 和绘制 1.1 契约不变。
- Windows、macOS、Linux 的 x86-64 与 ARM64 六个正式目标继续原生验证，不以交叉编译
  结果代替目标运行器上的窗口、绘制、无障碍和候选回放。
- 0.9 不新增公开表面；0.1.0 起的已发布签名均通过机器兼容检查。

## 安装示例

```toml
[依赖]
言界 = { 包 = "yanxu-ui", git = "https://github.com/yanxulang/yanxu-ui.git", 修订 = "v0.9.0", 版 = "^0.9" }

[权限]
图形界面 = true
原生扩展 = true
剪贴板 = true
文件对话框 = true
```

随后运行`yanbao 更`与`yanbao 查`，并在每个正式目标上重新生成自己的锁文件。

## 升级方式

从 0.8.x 升级只需把依赖修订改为`v0.9.0`、约束改为`^0.9`，重新生成各目标锁并运行应用
验收；公开 API 没有变化。从更早版本升级时，按迁移指南处理言台 1.0 锁、显式权限、托管
资源所有权、结构化关闭报告和 0.8 起的逐项无障碍语义。

不要跨目标复制锁，也不要只替换言台动态库。生产应用应保存依赖锁、目标系统冒烟结果、
标签 CI 链接和 Release 附件摘要。

## 制品与校验

Release 附带`yanxu-ui-0.9.0-six-targets.tar.gz`、同名`.sha256`、完整清单快照和 API JSON。
六目标候选覆盖包入口、15 个公开示例、14 项单元测试、11 条无窗口集成、5 条真实窗口路径、
4096 轮交互、1024 轮资源归零和 Linux x86-64 Release 性能预算。

汇总后的独立演练会产生`release-candidate-report`，记录来源提交、归档／清单／API 摘要和
六个自动关闭 YXB 的摘要与字节数。正式 Release 只复用与标签提交一致且已经通过全部门禁的
标签 CI 候选，不重新构建二进制。

## 第三方依赖

言台 1.0.0 采用 MIT OR Apache-2.0 双许可，言据 1.1.2 采用 MIT 许可。言界不含自有
DLL、dylib 或 so；平台原生依赖由言台承担。仓库使用的 checkout、upload-artifact 和
download-artifact 动作均固定到 40 位提交。

## 已知限制

- 数据源加载器仍为同步回调；异步调度、网络权限和取消由应用负责。
- 可见列表项离开当前语义树后，旧编号不能用于后续辅助动作。
- 菜单快捷键字段只负责显示；全局快捷键由窗口快捷键表注册。
- GPU、移动端和 Web 后端不属于 0.9 的支持范围，当前桌面目标统一使用言台 CPU 二维后端。
