# 文本编辑与中文 IME

`文本/文档.yx`以 Unicode 字符位置保存正文、光标和锚点；不会把 UTF-8 字节位置误当成字符
位置。言台文字命中和 IME 组合选区使用 UTF-8 字节位置，边界处通过`字节转字符`和
`字符转字节`显式转换。

已实现：

- 单行和多行正文；
- 光标、正向／反向选区、全选和鼠标拖选；
- 插入、退格、向前删除；
- 复制、剪切、粘贴；
- 单步撤销、重做和 256 项历史上限；
- 左右、上下、Home、End 与 Shift 扩展；
- 多行横纵滚动；
- IME 开始、更新、提交、取消及组合选区；
- 候选窗跟随当前显示光标区域。

输入框获得焦点时调用言台启用文本输入，并在光标变化后更新候选区矩形；失焦时取消组合
并停止输入法。按键事件只做命令和导航，实际文字由`文本输入`或`IME组合提交`插入，因此
拼音、五笔、日文、韩文、Emoji 和扩展 Unicode 不依赖键盘布局猜测。

辅助技术的`设置值`同样经过 65536 UTF-8 字节上限后替换文档并触发变化事件；`选择`参数
使用 UTF-8 字节位置，起点和终点都必须往返到精确字符边界，不能落在多字节字符内部。
`复制`和`粘贴`复用与键盘路径相同的权限化系统剪贴板接口。

0.1 的选区背景对单行和常见同一行选区给出精确宽度；复杂跨行选区仍使用保守矩形，后续
版本会直接消费言台整形行的原文索引映射生成逐行几何。
