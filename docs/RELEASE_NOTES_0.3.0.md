# 言界 0.3.0

言界 0.3.0 在已发布的言台 1.0 稳定契约上补齐常用桌面表单。新增控件共享有界状态、
键盘与指针交互、主题反馈以及原生无障碍动作回传，并继续由同一份言序源码覆盖六个正式
目标。

## 新功能

- 复选框支持二态、混合态、空格键、指针取消、变化事件和原生点击动作。
- 单选框提供非空组约束、同组互斥、Tab 收集规则、方向键循环和原生选择动作。
- 切换控件支持鼠标、空格、回车、按下反馈、检查语义与原生点击动作。
- 下拉选择支持有界选项、指针弹框、滚轮、方向键、Home/End、确认、取消和首字匹配；
  辅助技术可用有效整数索引或精确文字设置值。
- 横向/纵向滑块支持有限范围、步长吸附、拖动捕获、完整键盘操作和设置值/增减动作。
- 进度条支持确定值和不定忙碌状态，始终保持不可聚焦、无动作的输出语义。
- 新增只读、必填和无效通用表单状态，以及完整表单指南和可构建示例。

## 修复内容

- 只读输入框现在允许焦点、选择和复制，但拒绝文字输入、IME 修改、删除、剪切、粘贴、
  撤销/重做及辅助技术设置值；切换到只读会取消已有 IME 组合。
- 表单构造先验证组、选项和数值范围，失败时不会把半初始化控件留在父容器。
- 下拉选项替换、只读切换和关闭路径会折叠弹框并释放指针捕获；滑块状态切换同样结束拖动。
- 下拉选项、滑块范围、步长、进度跨度和辅助技术参数都经过容量、有限性和原子更新校验。
- 表单内部实现从包入口拆出，避免完整 Release 字节码超过单模块大小上限。

## 兼容性与平台

- 最低源码工具链仍为言序 1.1.9，正式候选继续由言序 1.1.20 在隔离源码副本中构建回放；
- 依赖继续固定言台 `v1.0.0` 和言据 1.1.2；ABI v2、平台 1.7、事件 1.3、无障碍 1.0
  与绘制 1.1 均未改变；
- Windows、macOS、Linux 的 x86-64 与 ARM64 六目标继续使用同一言序源码；
- 0.3 对 0.1/0.2 公开 API 只做增加，不删除或改变既有方法、事件和主题字段。

## 安装示例

```toml
[依赖]
言界 = { 包 = "yanxu-ui", git = "https://github.com/yanxulang/yanxu-ui.git", 修订 = "v0.3.0", 版 = "^0.3" }

[权限]
图形界面 = true
原生扩展 = true
剪贴板 = true
文件对话框 = true
```

随后运行`yanbao 更`与`yanbao 查`，并重新构建应用 Bundle。

## 升级方式

把依赖修订更新为`v0.3.0`、版本约束更新为`^0.3`，再为每个目标重新生成锁文件。应用可
逐步采用新增控件；原有按钮、输入、布局、主题和无障碍代码无需迁移。表单业务校验继续
由应用维护，并通过`必填`和`无效`状态反馈，不从绘制颜色反推业务状态。

## 制品与校验

Release 附带`yanxu-ui-0.3.0-six-targets.tar.gz`、同名`.sha256`、
`yanxu-ui-0.3.0.toml`完整包清单和 API JSON 快照。六目标候选分别验证 12 个公开示例与
包入口共 13 个 YXB；归档保留目标锁、由言序 1.1.20 构建并实际回放的自动退出窗口 YXB
及运行时元数据。Release 只复用标签同一提交已经通过的 CI 候选。

## 第三方依赖

言台 1.0.0 采用 MIT OR Apache-2.0 双许可，言据 1.1.2 采用 MIT 许可。0.3.0 不新增直接
原生依赖；窗口、绘制、UIA、NSAccessibility 与 AT-SPI 继续由锁定的言台制品提供。

## 已知限制

- 下拉弹框的窗口边缘自动翻转、统一叠层排序、子菜单和模态焦点陷阱属于 0.4 范围；
- 列表、标签页和菜单已有基础角色，逐项目集合语义与异步数据源将在后续小版本补齐；
- 多行跨行选区背景仍使用保守矩形，文本值、IME 和 UTF-8 无障碍选区不受影响；
- GPU 后端不是 0.3 的范围，当前统一使用言台 CPU 二维后端。
