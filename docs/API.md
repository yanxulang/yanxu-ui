# 言界 0.6 API

以下接口与 `src/主.yx` 的公开声明一致。

叠层、菜单和弹出层见[叠层与菜单指南](MENUS.md)；文本和数据视图见[文本与数据视图](DATA_VIEWS.md)；
帧动画、背压和诊断见[动画、帧反馈与运行诊断](ANIMATION_AND_DIAGNOSTICS.md)；
表单状态、交互边界和完整组合示例见[完整表单指南](FORMS.md)。

## 应用

```yanxu
定 应用 为 界面.应用（「名称」）；
定 窗口 为 应用.窗口（{「标题」：「主窗」，「宽」：800，「高」：600}）；
应用.主题（「浅色」）；             # 或「深色」
应用.主题配置（配置典）；
应用.更新主题（局部覆盖）；         # 原子失效缓存并返回新修订
应用.定时器（1000，真，回调）；
应用.加载图片（PNG或JPEG字节）；
应用.诊断快照（）；               # 调试快照（）为兼容别名
应用.运行（）；应用.退出（）；应用.关闭（）；
```

`监听（类型，回调）`可处理应用级事件；未处理的`退出请求`默认结束事件循环。

## 窗口

```yanxu
窗口.行（配置）；窗口.列（配置）；窗口.堆叠（配置）；窗口.网格（配置）；
窗口.显示（）；窗口.隐藏（）；窗口.标题（「新标题」）；窗口.大小（宽，高）；
窗口.同步无障碍（）；窗口.无障碍快照（）；
定 任务 为 窗口.动画（配置，更新回调）；窗口.取消动画（任务）；
窗口.诊断快照（）；
窗口.关闭时（回调）；窗口.快捷（「主+s」，回调）；窗口.关闭（）；
```

`同步无障碍`只在语义或布局脏时提交完整树；`无障碍快照`先同步再返回言台规范化树和
修订摘要。窗口内部还公开`根（）`、文字测量／命中、输入法配置和失效接口，供自定义控件
使用。

动画配置接受有限`起值`／`终值`、1–3600000 的`时长毫秒`、四种`缓动`和可选`完成`
回调。返回任务可查询、取消和取得无回调对象的快照；更新由匹配的真实`帧呈现`反馈推进，
每窗最多 1024 个活动动画且始终只有一个待呈现帧。应用和窗口诊断是版本化、可 JSON
序列化的内容安全摘要，详见动画与诊断指南。

## 控件通用接口

```yanxu
控件.显示（真）；控件.启用（真）；控件.可聚焦（真）；
控件.只读（假）；控件.必填（假）；控件.无效（假）；
控件.尺寸（宽，高）；控件.最小尺寸（宽，高）；控件.最大尺寸（宽，高）；
控件.外边距（【上，右，下，左】）；控件.伸展（1）；控件.样式（样式典）；
控件.监听（「事件类型」，回调）；控件.捕获（「事件类型」，回调）；
控件.可访问名称（「名称」）；控件.可访问描述（「描述」）；控件.关闭（）；
```

`框`是布局后的逻辑像素矩形。`语义（）`返回带稳定正整数`编号`、逻辑像素边界、状态、
操作和子树的言台无障碍协议节点；同一控件存活期间编号不变。`查找编号（编号）`在当前
子树重新解析协议节点身份，不缓存已经关闭的控件对象。自定义控件可覆写
`执行无障碍动作（动作，参数）`，只对当前可交互状态和自己声明的操作返回真。

`只读`、`必填`和`无效`是表单通用状态；只有言台无障碍协议允许的输入、可检查、组合框和
滑块角色才把它们写入语义树。只读输入框仍允许焦点、选择、复制和程序主动更新内容，但会
拒绝键盘删除、文字输入、IME 组合、剪切、粘贴及辅助技术设置值。

## 容器和控件

| 创建方法 | 返回能力 |
| --- | --- |
| `行／列／堆叠／网格（配置）` | 嵌套容器 |
| `文字（内容）` | `内容／取内容` |
| `按钮（内容）` | `内容／取内容／点击`及键盘激活 |
| `复选框（内容）` | 二态/混合态、鼠标/空格键激活与`变化时` |
| `单选框（内容，组）` | 同组互斥、方向键循环导航、选择语义与`变化时` |
| `切换（内容）` | 开/关状态、鼠标/空格/回车激活与`变化时` |
| `下拉选择（文字列）` | 有界选项、弹框/滚轮/首字导航、`选择／取选择／取值／变化时` |
| `滑块（配置）` | 横向/纵向拖动、键盘与`值／范围／步长／方向／变化时` |
| `进度条（配置）` | 有界确定进度、不定忙碌状态与`值／范围／不定` |
| `输入框（配置）` | 单行编辑、只读/必填/无效语义、`内容／取内容／选择范围／变化时` |
| `多行输入框（配置）` | 多行编辑与滚动 |
| `滚动（配置）` | 子控件滚动容器 |
| `列表（文字列）` | 虚拟行绘制、选择和变化事件 |
| `虚拟列表（数据源，配置）` | 可见范围加载、有限缓存、增量更新、加载状态和无障碍选择 |
| `标签页（标签列）` | 预置标签；`页（名称）`创建页面 |
| `分割面板（配置）` | 两个子面板和可拖动分隔条 |
| `菜单（项目列）` | 有界项目、子菜单、指针／键盘导航、选择事件和无障碍动作 |
| `弹出层（配置）` | 锚定、自动翻转、外部点击关闭、Escape 与 `打开／关闭弹出` |
| `画布（配置）` | `命令／清空命令`追加言台绘制命令 |
| `图片（资源）` | 图片测量和绘制 |

配置常用字段为`可见`、`启用`、`可聚焦`、`首选宽`、`首选高`、`最小宽`、`最大宽`、
`外边距`、`伸展`和`样式`。容器另支持`内边距`、`间距`、`主对齐`和`交对齐`。

下拉选择最多接受 4096 个选项，每项最多 65536 UTF-8 字节。`选项（新列）`原子替换选项并
尽量保留当前值；`展开／折叠／取展开`控制弹框。辅助技术的`设置值`接受有效整数索引或
精确文字值，只读和禁用状态拒绝交互改值；程序调用`选择（序）`仍可主动更新状态。

滑块配置使用有限数`最小`、`最大`、`步长`和`值`，要求最大值大于最小值且步长大于零；
`方向`为`横向`或`纵向`。所有指针、键盘和辅助技术输入都会夹取并对齐到步长；方向键、
Home/End、PageUp/PageDown 以及`设置值／增加／减少`共享同一变化路径。只读状态拒绝交互
调整，但`值（新值）`、`范围（最小，最大）`和`步长（新值）`仍可由程序主动调用。

进度条配置使用有限数`最小`、`最大`和`值`，并要求有限跨度；值始终夹取到范围。它是
不可聚焦、无动作的输出控件，`不定（真）`会隐藏确定值并在语义树公开`忙碌`，恢复确定
状态后继续使用最后一次程序设置的值。言台协议不允许进度条声明只读、必填或无效状态，
因此这些通用样式状态不会被写入进度条语义节点。

### 文本与数据视图

`文本.文档`提供`行数`、`行首索引`、`行尾索引`、`行文本`、`选区行段`、`设置字节选区`
和`选区字节范围`。`多行输入框`消费这些范围，在视口内按行生成文字和选区命令。

`数据源`是独立导出模块；`数据源（总数，加载器）`的每次范围加载最多 256 项，缓存默认
1024 项。`虚拟列表`只请求可见范围，并公开`刷新数据`、`数据状态`、`数据错误`、`数据数量`
和`设置行渲染器`。详见[数据源 API 参考](DATA_SOURCE_API.md)。

数据源监听返回单调递增编号：保存`监听（事件名，回调）`的结果，并用
`取消监听（事件名，编号）`精确解除对应订阅；不要依赖对象方法值比较。

## 系统服务

包级函数直接保持言台的统一行为：`剪贴板读取`、`剪贴板写入`、`打开文件`、
`打开多个文件`、`保存文件`、`选择目录`和`选择多个目录`。
