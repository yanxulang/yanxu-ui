# 安全与权限

言界不加载自有原生代码；原生边界全部由固定版本言台包承担。应用仍须显式声明：

- `图形界面`：创建和运行原生窗口；
- `原生扩展`：加载言台 ABI v2 制品；
- `剪贴板`：仅在调用复制／剪切／粘贴时使用；
- `文件对话框`：仅在调用公开对话框函数时使用。

言界不会因为图形权限自动获得网络、进程、文件系统任意访问、通知或托盘权限。主题配置
通过言据模式验证，主版本不符、继承循环和损坏数据都会显式失败。高频事件与绘制帧受言台
队列和二进制协议上限约束。

无障碍焦点和动作事件在执行前再次核对当前树修订、稳定节点编号、控件可交互状态、动作
名称和参数。设置值内容不写入诊断；选择范围必须是当前 UTF-8 文本的精确字节边界。旧树
请求、已关闭控件和畸形参数会被忽略，不会复用为普通控件事件。

下拉选择在进入控件树前验证选项：最多 4096 项，每项必须是文字且不超过 65536 UTF-8
字节。对外返回选项副本；辅助技术只能用有效整数索引或精确文字改值，不能绕过只读、
禁用或关闭状态。

滑块在加入控件树前验证有限范围、严格递增的上下界、正步长和协议方向。指针坐标、键盘
增量及辅助技术数值统一经过夹取和步长对齐；非数、无穷、错误类型和只读/禁用状态都不会
改变现有值。

进度条同样拒绝非有限边界、溢出跨度和非有限值，并把值夹取到有效范围。它始终不可聚焦
且不声明动作；不定状态只公开空值和忙碌标记，不伪造数值进度或可编辑表单状态。

动画只接受有限起终值、有限跨度、1–3600000 毫秒时长和四种已知缓动；每窗活动任务限制
为 1024，通用队列构造上限为 4096，编号在安全整数空间耗尽时拒绝继续分配。更新和完成
回调的错误隔离到所属任务，回调内取消或窗口关闭会先清理任务与待呈现帧，不形成无界任务
或帧队列。

应用和窗口诊断只返回版本、编号、状态及有界计数。快照不包含窗口标题、显示器名称、控件
文字、输入内容、密码、回调对象、无障碍文字树或原生句柄；应用监听类型名称最多返回 256
项。平台资源不可用时，诊断返回稳定错误摘要，不重新访问已释放资源。

报告漏洞时请使用 GitHub 私密安全公告，不要在公开 Issue 中附加利用代码或用户数据。
